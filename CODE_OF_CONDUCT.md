# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) (v2.1).

**In short:** be respectful, be constructive, assume good faith. Harassment, discrimination, and personal attacks aren't tolerated.

Report concerns by opening a private security advisory or contacting the maintainer via their [GitHub profile](https://github.com/justsima). Maintainers may remove, edit, or reject contributions that violate this code.
